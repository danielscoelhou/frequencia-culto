# 📊 Sistema de Registro de Frequência de Culto

Aplicação web offline desenvolvida para registrar a frequência de pessoas em cultos de igreja, separando membros e visitantes por categorias.

---

## 🚀 Funcionalidades

* Registro de frequência por categorias:

  * Crianças
  * Intermediários
  * Adolescentes
  * Jovens
  * Senhoras
  * Varões
    
* Separação entre membros e visitantes
  
* Cálculo automático de:
  * Total de membros
  * Total de visitantes
  * Total geral
    
* Campo de observações (nomes de visitantes ou ocorrências)
* Geração de relatório formatado
* Copiar relatório com um clique
* Funcionamento **100% offline**
* Instalável como **aplicativo (PWA)**

## 📦 Como usar

1. Acesse o sistema pelo navegador
2. Preencha os dados do culto
3. Clique em **Gerar Relatório**
4. Clique em **Copiar Relatório**
5. Cole no WhatsApp, relatório ou planilha

---

## 📲 Instalar no celular

1. Abra o sistema no navegador (Chrome)
2. Clique em **Adicionar à tela inicial**
3. Use como aplicativo offline
